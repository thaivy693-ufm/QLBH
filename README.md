Hệ Thống Quản Trị Nghiệp Vụ Baya (SQL-based)
📌 Tổng quan dự án
Dự án thiết kế và triển khai cơ sở dữ liệu quản trị cho mô hình kinh doanh nội thất, tập trung vào sự chính xác của dữ liệu và luồng chứng từ Kế toán - Kho.

🛠 Kỹ năng Kỹ thuật & Nghiệp vụ
Cơ sở dữ liệu: MySQL (Thiết kế cấu trúc 5 bảng chuẩn hóa).

Nghiệp vụ ứng dụng: Quản lý danh mục hàng hóa, đối soát đơn mua hàng (PO), phiếu nhập kho và xuất hóa đơn VAT (M-Invoice).

Kỹ năng SQL: Viết các truy vấn báo cáo doanh thu, tính toán tổng tiền từ bảng chi tiết bằng INNER JOIN, GROUP BY, SUM.
